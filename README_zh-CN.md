# 适用于 Spigot 的 PotatoCore

##### [English](https://github.com/dmzz-yyhyy/PotatoCore/blob/main/README.md) | [中文 (简体)](https://github.com/dmzz-yyhyy/PotatoCore/blob/main/README_zh-CN.md) )

## 功能

- [x] 上线/离线广播
- [x] QQ和游戏聊天转发消息
- [ ] 从 QQ 执行命令（仅限管理员）
- [ ] 管理材料仓库
- [ ] 每日进度提醒

## 用法

### 在游戏中

| 命令          | 功能       |
|-------------|----------|
| /ptb reload | 重新加载配置文件 |

### 组内

| 命令     | 用法         | 权限     |
|--------|------------|--------|
| !#ping | 检查服务器状态    | 无      |
| #[消息]  | 转发消息到服务器聊天 | 无      |

## 权限

| 权限           | 注意             | 默认  |
|--------------|----------------|-----|
| potato.admin | PotatoCore 管理员 | OP  |
