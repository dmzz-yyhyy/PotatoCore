# PotatoCore for the UnitedPotato

A multifunctional plugin (not yet).

PotatoCore runs on the UnitedPotato Minecraft Server.

##### [中文 (简体)](https://github.com/dmzz-yyhyy/PotatoCore/blob/main/README_zh-CN.md) | [中文 (繁體)](https://github.com/dmzz-yyhyy/PotatoCore/blob/main/README_zh-Hant.md) | English

## Features

- [x] In-game players online / offline broadcasting to QQ group
- [x] Forward messages between QQ and in-game chat
- [x] Execute commands from QQ Group (admins only)
- [x] Reload config file using command
- [x] Daily progress reminder
- [ ] Custom command(on qq group)
- [ ] Manage storage items
- [ ] Express box

## Usage

### In-game Commands

| Command       | Description                |
|---------------|----------------------------|
| /ptb reload   | Reload configuration files |

## Permissions

| Permission      | Note             | Default |
|-----------------|------------------|---------|
| potato.admin    | PotatoCore admin | OP      |
